# ♟️ EWA Clothing | Admin Dashboard

**👗 EWA Clothing Admin Suite:** A high-end monochrome interface for Pakistani fashion brands, featuring a 35+ item dynamic inventory, PKR currency localization, TCS/Leopard logistics tracking, and smooth CSS transitions—fully optimized for Vercel deployment. 🚀✨

---

## 💎 Key Features
* **Monochrome UI:** High-end aesthetic designed for premium fashion houses.
* **Smart Inventory:** 35+ product catalog with automated "Low Stock" indicators.
* **PKR Localization:** Built-in support for Pakistani currency and regional sales.
* **Logistics Ready:** Integrated UI for TCS and Leopard courier tracking.
* **Vercel Optimized:** Clean directory structure for instant CI/CD deployment.

## 🛠️ Tech Stack
* **Frontend:** HTML5, Modern CSS3, JavaScript (ES6+)
* **Framework:** Bootstrap 5
* **Animations:** Animate.css
* **Icons:** Bootstrap Icons
